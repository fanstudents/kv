# Agent 角色設計系統（Mascot Design System）

> 適用範圍：agent-team 銷售頁、後台 Avatar、未來所有新 Agent 的角色創作。
> 原則：**角色是「配方」，不是「插畫」**——新增角色時不重新發想，只填配方。

## 角色配方

```
角色 = 物種（部門決定） × 主色（agent-data 色盤） × 職務道具（一個） × 統一臉型
```

| 欄位 | 規則 | 範例 |
|------|------|------|
| 物種 | 由部門決定，同部門同物種 | 客服部 → 兔 |
| 主色 | 沿用 `src/lib/agent-data.ts` 既有色盤，一隻一色 | `#EC4899` |
| 道具 | 一個，代表職務，放右下角白色泡泡 | 🎧 |
| 臉型 | 固定不變：圓頭、點點眼＋白色高光、微笑（鳥／貓頭鷹改小嘴喙）、白色腮紅 | — |

## 物種對照表（部門 → 物種 → 個性理由）

| 部門 | 物種 | 理由 | 特徵差異 |
|------|------|------|----------|
| 客服部 | 兔 | 耳朵長、聽得多、反應快 | 直立長耳（內耳淺色） |
| 行銷部 | 鳥 | 把聲量傳出去 | 頭頂羽冠＋橘色小嘴喙 |
| 營運部 | 熊 | 穩重可靠、扛得住 | 圓耳（內耳淺色） |
| 財務部 | 貓頭鷹 | 精明、夜間也盯著數字 | 耳羽＋白色大眼圈＋嘴喙 |
| 人資部 | 狗 | 親人、忠誠 | 頭側垂耳（深一階色） |
| 情報部 | 貓 | 神出鬼沒、觀察力強 | 三角耳（內耳淺色）＋鬍鬚 |

新部門 = 挑一個新物種（先想「這個部門的個性像什麼動物」），其餘全部沿用統一臉型規則。

## 統一臉型規格（SVG，viewBox 0 0 64 64）

- 頭：圓形 `cx=32 cy=38 r=19`，填滿主色，無外框線
- 眼：兩個深棕點 `#402E32 r=2.6`（位置 25/39, y=37）＋白色高光小點
- 嘴：微笑弧線 `M29 42.5 q3 2.8 6 0`（鳥類改橘色三角嘴喙 `#F6A623`）
- 腮紅：白色半透明橢圓 `rgba(255,255,255,.4)`
- 道具泡泡：白圓 `cx=49 cy=51 r=8.8`＋emoji 置中
- 物種特徵一律只動「耳朵／頭頂」區域，臉部不變

實作：`marketing/agent-team-landing.html` 內的 `mascotSVG(species, color, prop)` 函式，
之後可直接移植成後台的 React `<Mascot>` 元件取代 `Avatar.tsx` 的照片版。

## 高精度插畫版（廣告素材／LINE 貼圖用）AI 生圖 Prompt 模板

```
A cute round mascot character, flat vector illustration style, kawaii,
species: {兔/鳥/熊/貓頭鷹/狗/貓}, body color {HEX},
simple dot eyes with white highlight, small smile, white blush cheeks,
holding a {道具}, plain white background, sticker style,
consistent with LINE Friends proportions, no outline, soft shadows
```

規則：一次只換 `{species}`、`{HEX}`、`{道具}` 三個變數，其他描述不動，確保全系列一致。

## Do / Don't

- ✅ 新角色一律先填配方表，再產圖
- ✅ 個體差異只靠「主色＋道具」；物種內所有個體共用同一張基底
- ❌ 不要為單一角色改臉型比例、眼睛樣式或筆觸
- ❌ 不要在同一部門混物種
- ❌ 道具不超過一個（識別度來自單一明確符號）
