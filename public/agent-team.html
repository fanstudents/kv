為什麼是「部署」而不是「訂閱雲端」

別人的 AI 住在雲端，
你的 AI 住在你公司。

客戶名單、訂單、營收數字，是你最不該交出去的東西。Agent Team 整套系統部署在「你的環境」——你的主機、你的私有雲、你的資料庫。我們負責讓它上工，資料的鑰匙一直在你手上。

SEC-01

部署在你的環境

地端主機、公司 NAS、或你自己的雲端帳號皆可。系統跑在你的基礎設施上，不是我們的多租戶雲。

對話紀錄、客戶名單、訂單資料全部落在你的資料庫。要備份、要搬家、要刪除，都是你說了算。

常見問題

老闆們最常問的五件事

我們公司沒有工程師，可以導入嗎？

可以。部署由我們全程處理（線上或到府），日常使用只需要會用後台介面——像用 LINE 一樣簡單。教育訓練包含在部署費裡，教到會為止。

沒有自己的 AI API Key 也能用嗎？

能。預設方案由我們代管 AI 用量，月費全包不另計。等你想更省，隨時可以切換成自己的 API Key，各級月費最低再降 40%。

可以只聘一個 Agent 試試看嗎？

可以，一位起聘。多數客戶從客服或訂單 Agent 開始，第一個月看到值班紀錄後，再逐步增聘其他部門。

部署要多久？會不會影響現有系統？

標準部署 1–3 個工作天。系統以旁掛方式串接你現有的 LINE、官網與後台，不改動、不取代你原本的系統，隨時可整套下線。

資料真的不會外流嗎？

系統與資料庫都建在你指定的環境（地端或你的雲端帳號），對話與客戶資料不經過我們的伺服器存放。搭配權限分級與完整稽核紀錄，每一筆存取都有跡可循。
